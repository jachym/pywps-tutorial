Welcome to PyWPS Course!
========================

This document is intended to be used as course and workshop tutorial. It is
written for PyWPS 3.x version. If you consider any chapter of this material
unclear, please `contact us <http://pywps.wald.intevation.org/community/index.html>`_

What is PyWPS
-------------
`PyWPS <http://pywps.wald.intevation.org>`_ is implementation of `OGC Web
Processing Service <http://opengeospatial.org/standards/wps>`_ standard on
the server side.

.. note:: For this tutorial, we have deployed example server, at http://apps.esdi-humboldt.cz/pywps/
