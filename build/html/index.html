This document is intended to be used as course and workshop tutorial. It is written for PyWPS 3.x version. If you consider any chapter of this material unclear, please contact us

What is PyWPS¶

PyWPS is implementation of OGC Web Processing Service standard on the server side.

Note

For this tutorial, we have deployed example server, at http://apps.esdi-humboldt.cz/pywps/
